# Expense Tracker

A simple and responsive Expense Tracker web app to manage income and expenses with a clean and user-friendly interface.

## 🚀 Features

* Add and delete transactions
* Track income and expenses
* View total balance instantly
* Clean and responsive UI

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## 📂 Project Setup

1. Clone the repository
2. Open `index.html` in your browser
